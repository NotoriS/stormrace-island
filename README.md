# Stormrace Island

Welcome to **Stormrace Island**, a 3D platformer where players must navigate an island and reach their campsite before a raging storm engulfs them! This game was developed as part of **Brackeys Game Jam 2024.2**.

[Play Stormrace Island on Itch.io](https://notoris.itch.io/stormrace-island)

## About the Game
In **Stormrace Island**, you race against time as severe weather builds and threatens to trap you in a deadly storm. You'll start your adventure in clear weather, but as you race towards your campsite, you'll experience worsening weather conditions like cloud cover, rain, lightning, and crashing waves. Your goal is to navigate through the island's terrain and make it back to your camp before the storm overtakes you!

### Game Jam
This game was created for [Brackeys Game Jam 2024.2](https://itch.io/jam/brackeys-12), an event where game developers are challenged to create a game in only one week.

## Features
- **Dynamic Weather**: Experience changing weather with increasing intensity as you progress, including cloud cover, rain, lightning, and intense waves.
- **Race Against Time**: Complete the race before time runs out and the storm catches up to you.
- **Coin Collection**: Challenge yourself to pick-up as many coins as you can during your journey to the campsite.
